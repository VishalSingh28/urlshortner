
<br>

## Production
```bash
$ docker-compose build && docker-compose up
```

## Development
#### Server:
```bash
$ cd server && npm
```
```bash
$ npm build:watch
```
```bash
$ npm dev
```
#### Client:
```bash
$ cd client && npm
```
```bash
$ npm start
```
